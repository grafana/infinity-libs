# Contributing Guide

## Pre-requisites

- Git
- NodeJS 20+
- Yarn
- go 1.22
- Mage
- Docker

## How to run this locally

- Clone the repo locally `git clone https://github.com/yesoreyeram/grafana-plugins yesoreyeram-grafana-plugins` and `cd yesoreyeram-grafana-plugins`
- Install packages `yarn`
- Test `yarn test`
